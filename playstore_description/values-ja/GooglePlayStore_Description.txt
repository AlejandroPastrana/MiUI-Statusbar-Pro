####---#  Play Store full description
MiUI Statusbar Proを使用することで、MIUIステータスバーのスタイルを変更することができます。スタイルの変更にXposedフレームワークは不要で、MIUIテーマとも競合しません。
これはすべて、よく知られている"Center Clock Mod"から始まり、今では7つの異なるデザインの中から選択することができます！

特徴と機能:
**MiuiSystemUI.apkのバックアップを作成
**MiuiSystemUI.apkのバックアップを復元
**MIUIステータスバーのスタイルを7種類のデザインに変更
**MIUIがインストールされた非Xiaomi端末でも動作します！
**Xposedは必要ありません！
 
前提条件:
**アプリの動作にはroot権限が必要
**MIUIファームウェアでのみ利用可能
